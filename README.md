# Como executar o projeto

## Pré-requisitos

npm / yarn

```bash
# clonar repositório
git clone https://github.com/MarceloDevCruz/portfolio-reactJs portfolio

# entrar na pasta do projeto
cd portfolio

# instalar dependências
npm install

# executar o projeto
npm run dev

# acessar porta 5173 do localhost por algum navegador
locahost:5173
```
